1. Introduction


The main aim of the project is to design an appropriate webpages. The main page that is index page is the landing page from where six different webpage is accessed. Each page containing the header and footer and some of the page contain the sidebar too. Hyper link is used to connect each page. All the pages contain the navigation bar from where user can smoothly jump into the page that the user desire. The theme of the project is to use maximum number of code to design user friendly and eye catchy webpage.

Targeting the visitors most of the people use smart phone. Almost 60 percent use the smart phone so the designed is most in priority for the mobile or the smart phones. All the design is media friendly and the eye friendly. Each web page looks good in mobile to personal computer.

The main index.html page is the landing page for the user from where all the other page can be accessed. User cannot directly get accessed other page without opening index page but once the home page is opened the navigation on each page help to jump towards next page.

The about.html page is the second web page in the project which contain the description of the author that means mine information along with my experience as a developer. In this page the header, navigation and footer is as same as of index page.

The third page is the social media page which contain the hyper link that directly link to my social site such as Facebook, Twitter, Instagram and Linked in. Image is used for the area to link to social site.

The CV page contain the CV of the developer. In this page there is two division, one is for developer’s CV where as other is to create the CV of the user. To create the CV form is used. In this page the ad position is also available. This page also contain header, navigation and footer section.

The next page that is the fifth page is models GP page which contain the report of the project. This contain the experience, introduction, wireframe and mockup. This is totally used for the reporting.

The final or the last page is contact page which contain the form. If the user will to give comment then it is possible from this page as it contain form including name, email and the comment they wanted to comment.
